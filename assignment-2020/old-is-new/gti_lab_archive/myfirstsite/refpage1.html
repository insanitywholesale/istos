Installing Arch Linux can be near automatic

These are my two scripts for before and after the chroot step

#!/bin/bash

timedatectl set-ntp true

mkfs.fat -F32 /dev/sda1

mkfs.ext4 /dev/sda2

mkswap /dev/sda3

swapon /dev/sda3

mount /dev/sda2 /mnt

mkdir /mnt/boot

mount /dev/sda1 /mnt/boot

cp /etc/pacman.d/mirrorlist /etc/pacman.d/mirrorlist.backup

rankmirrors -n 9 /etc/pacman.d/mirrorlist.backup > /etc/pacman.d/mirrorlist

pacstrap /mnt base base-devel

genfstab -U /mnt > /mnt/etc/fstab

arch-chroot /mnt

That was the first part. Now, on to the second one.

ln -sf /usr/share/zoneinfo/Europe/Athens /etc/localtime

hwclock --systohc

echo 'en_US.UTF-8 UTF-8' >> /etc/locale.gen

echo 'LANG=en_US.UTF-8' > /etc/locale.conf

echo 'archaos' > /etc/hostname

echo '127.0.0.1 localhost' > /etc/hosts

echo '::1 localhost' >> /etc/hosts

echo '127.0.1.1 archaos.hell archaos' >> /etc/hosts

mkinitcpio -p linux

pacman -S intel-ucode grub efibootmgr --no-confirm

grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=archaos

grub-mkconfig -o /boot/grub/grub.cfg

pacman -S vim i3 i3-lock i3-status xorg xorg-xinit qemu libvirt firewalld dnsmasq ovmf polkit-gnome openssh openssh-askpass virt-manager networkmanager network-manager-applet compton redshift feh dmenu termite firefox thunar ntfs-3g lxappearance qt5ct ttf-dejavu ttf-liberation noto-fonts pango ttf-font-awesome qt5-styleplugins rsync cpupower xfce4-screenshooter htop vlc gparted kdenlive thunderbird steam keepass pavucontrol ranger --no-confirm

useradd -m -g users -G wheel users kvm libvirt -s /bin/bash angle

echo 'remember to do passwd for angle and root'

Now you just need to set a password for root, and one for your user. That can be achieved using

passwd

and

passwd angle

respectively

Enjoy :)
